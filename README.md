# Brengine 3D

Full scale 3D game engine for personal proposes

![image1](https://github.com/user-attachments/assets/d1ee46cd-13f4-415a-bc5e-f1c702d21581)

# Controls

**W/A/S/D** - movement

**F5** - change render mode

**F2** - take screenshot

# Stack
- OpenGL 4.3
- GLEW
- GLFW
- GLM
- C99 std lib
- C++17 std lib
